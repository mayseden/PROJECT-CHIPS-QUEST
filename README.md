 Chips, Design, and Accessibility

A Responsive & Accessible Web Project

Project Overview

This project is a responsive and accessibility-focused website centered around chips (the food, not data).
While the topic is fun and familiar, the goal of this project was serious: to design and build a website that is usable, inclusive, and accessible across devices and user needs.

The site demonstrates how thoughtful front-end design choices can improve usability for all users, including those using assistive technologies.

 Project Objectives

Practice responsive web design for different screen sizes

Apply web accessibility principles (WCAG-aligned)

Build a clean, user-friendly interface using semantic HTML

Ensure the site remains usable and readable across devices

 Technologies Used

HTML5 (semantic structure)

CSS3 (responsive layouts and styling)

Media queries for mobile, tablet, and desktop views

Accessibility Features

This project intentionally incorporates accessibility best practices, including:

Semantic HTML elements for improved screen reader navigation

Clear visual hierarchy and readable typography

Sufficient color contrast for readability

Responsive layouts that adapt to different screen sizes

Keyboard-friendly navigation considerations

Accessibility was treated as a core design requirement, not an afterthought.
